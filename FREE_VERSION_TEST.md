# 無料版テストガイド

## 🎯 無料版の機能

### ✅ 実装済み

1. **YOLOv8自動ボール検出**
   - 完全自動（タップ不要）
   - 自動弾道表示
   - 飛距離・最高到達点・滞空時間
   - **1日3回まで**
   - **回数で色分け**
     - 1回目: 青
     - 2回目: 緑
     - 3回目: 赤

2. **使用回数制限**
   - 1日3回まで分析可能
   - 残り回数を表示
   - 上限に達するとダイアログ表示
   - 翌日0時に自動リセット

3. **広告視聴で復活**⭐
   - 3回使い切った後
   - 広告視聴で+2回復活
   - 1日1回まで

4. **カメラ撮影**
   - 動画撮影
   - プレビュー

---

## 📱 テスト手順

### 1. アプリをビルド・インストール

```
Android Studio → Run
```

### 2. ゴルフショット動画を準備

- スマホで撮影
- またはサンプル動画をダウンロード

### 3. 弾道分析をテスト

#### 1回目
```
「自動弾道表示」をタップ
  ↓
動画を選択
  ↓
「自動分析」ボタンをタップ
  ↓
✅ 分析開始
✅ 弾道が表示される
✅ 飛距離が表示される（yd）
```

#### 2回目
```
同じ手順で実行
  ↓
✅ 残り2回と表示される
```

#### 3回目
```
同じ手順で実行
  ↓
✅ 残り1回と表示される
```

#### 4回目（制限）
```
「自動分析」ボタンをタップ
  ↓
⚠️ 「本日の利用上限に達しました」ダイアログ
📺 「広告を見て復活」ボタンが表示される
💎 「プレミアムプランで無制限」案内
```

#### 広告視聴で復活
```
「広告を見る」ボタンをタップ
  ↓
📺 広告が表示される（TODO: AdMob統合後）
  ↓
✅ +2回復活
✅ 4回目・5回目が使える
✅ 弾道線は青に戻る
```

#### 6回目（完全制限）
```
「自動分析」ボタンをタップ
  ↓
⚠️ 「本日の利用上限に達しました」ダイアログ
❌ 広告ボタンは表示されない（1日1回まで）
💎 「プレミアムプランで無制限」案内のみ
```

## 🔧 デバッグ機能

### テスト用無制限モード（現在有効）⭐

**現在、無制限モードが有効になっています！**

```kotlin
// UsageManager.kt
private const val DEBUG_UNLIMITED = true  // ← 現在true
```

**動作:**
- 回数制限なし
- 何回でも分析可能
- 色分けは正常に動作（1回目=青、2回目=緑、3回目=赤）
- ダイアログは表示されない

**本番リリース時:**
```kotlin
private const val DEBUG_UNLIMITED = false  // ← falseに変更
```

---

### 使用回数をリセット

テスト中に回数をリセットしたい場合：

```kotlin
// UsageManagerに追加済み
usageManager.resetForDebug()
```

Android Studioのデバッガーで実行するか、
一時的にボタンを追加してリセット可能。

## ⚠️ 既知の問題

### YOLOv8モデルがない場合

```
⚠️ YOLOv8モデルが見つかりません
```

→ `app/src/main/assets/yolov8n.tflite`を配置してください

### モデルなしでも動作

- アプリは起動する
- 自動検出のみ使えない
- エラーメッセージが表示される

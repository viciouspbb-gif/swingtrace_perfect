# SwingTrace with AI Coaching

ゴルフボールの弾道を3D表示し、AIがスイングをコーチングするAndroidアプリ

## 機能
- **📊 SwingTrace弾道シミュレーター**
  - ボール初速・クラブ速度の入力
  - 打ち出し角・打ち出し方向（左右）
  - スピン量・スピン軸の傾き
  - スマッシュファクター自動計算
  - キャリー・最高到達点・滞空時間の表示
  - サイドビュー・トップビュー切り替え
- **🎬 高度な弾道アニメーション**
  - 2点タップで自動的に放物線を生成
  - ボールが軌道に沿って動くアニメーション表示
  - 白い弾道線＋オレンジの外側線
  - 光エフェクト
  - 飛距離自動計算（m / yd）
- **🏌️ スイング分析機能（SmoothSwing風）**
  - 動画上でスイング軌道を手描き
  - スイング速度の自動計算
  - バックスイング/ダウンスイング時間の測定
  - インパクトポイントの検出
  - スイングテンポの評価
  - スロー再生・コマ送り（0.25x〜2x）
- **飛距離・最高到達点・滞空時間の自動計算**
- **クラブ別プリセット**（ドライバー・7番アイアン・ウェッジ）
- **物理シミュレーション**（空気抵抗・マグナス効果を考慮）
- **カメラ撮影・動画再生機能**
- **マテリアルデザイン3 UI**

## 使い方

### 🎬 動画でボールの弾道を表示（メイン機能）
1. ホーム画面で「**動画再生**」ボタンをタップ
2. ゴルフショットの動画を選択
3. **ボールの位置をタップ**（1回目）
4. **着地点をタップ**（2回目）
5. **自動的に弾道が表示されます**
   - 白い弾道線＋オレンジの外側線
   - ボールがアニメーションで飛ぶ
   - 飛距離が表示される（m / yd）
6. **リセットボタン**で何度でもやり直せる

### 📊 弾道シミュレーター
1. ホーム画面で「**弾道シミュレーター**」ボタンをタップ
2. **パラメータ入力**
   - ボール初速・クラブ速度
   - 打ち出し角・打ち出し方向
   - スピン量・スピン軸
3. **プリセット選択**（オプション）
   - ドライバー / アイアン
4. **計算ボタンをタップ**
   - 詳細なショットデータ表示
   - スマッシュファクター自動計算

### 🏌️ スイング分析
1. ホーム画面で「**スイング分析**」ボタンをタップ
2. スイング動画を選択
3. **軌道を描画ボタン（✏️）をタップ**
4. クラブヘッドの軌道を指でなぞる
5. **分析結果が自動表示**
   - スイング速度・テンポ
   - バックスイング/ダウンスイング時間
   - インパクト速度

### 📹 動画撮影
1. ホーム画面で「**動画撮影**」ボタンをタップ
2. カメラでゴルフショットを撮影
3. 撮影した動画で弾道を表示

## トラブルシューティング

### Gradle Syncが失敗する
- インターネット接続を確認
- `File → Invalidate Caches → Invalidate and Restart`

### ビルドエラー
- JDK 17以上がインストールされているか確認
- `./gradlew clean` を実行後、再ビルド

### アプリが起動しない
- エミュレータのAPI Level 24以上を確認
- logcatでエラーログを確認

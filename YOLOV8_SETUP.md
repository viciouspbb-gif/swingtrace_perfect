# YOLOv8自動ボール検出セットアップガイド

## 🎯 機能

**完全自動でボールを検出して弾道を表示！**

- ✅ YOLOv8による自動ボール検出
- ✅ リアルタイム軌跡追跡
- ✅ 自動弾道計算・表示
- ✅ 推定飛距離表示（m / yd）
- ✅ 最高到達点・滞空時間
- ✅ グラデーション弾道線（緑→黄→オレンジ→赤）
- ✅ **完全無料・オフライン動作**

---

## 📦 必要なもの

### 1. YOLOv8モデルファイル

**ダウンロード先:**
https://github.com/ultralytics/assets/releases/download/v0.0.0/yolov8n.tflite

または

```bash
pip install ultralytics
yolo export model=yolov8n.pt format=tflite
```

### 2. モデルファイルの配置

ダウンロードした`yolov8n.tflite`を以下に配置：

```
app/src/main/assets/yolov8n.tflite
```

**assetsフォルダがない場合は作成:**
```
app/src/main/
├── java/
├── res/
└── assets/          ← 新規作成
    └── yolov8n.tflite  ← ここに配置
```

---

## 🚀 使い方

### 1. アプリを起動

### 2. 「自動弾道表示」を選択

### 3. ゴルフショット動画を選択

### 4. 「自動分析」ボタンをタップ

**タップ不要！完全自動！**

## ⚙️ 技術仕様

### YOLOv8
- **モデル**: YOLOv8n（nano）
- **入力サイズ**: 640x640
- **推論速度**: 30-60fps（GPU使用時）
- **精度**: 高精度ボール検出

### 弾道計算
- **方式**: ハイブリッド
  - YOLOv8で実際の軌跡を検出
  - 物理エンジンでスムージング
- **結果**: リアル + 滑らか

## 🔧 トラブルシューティング

### モデルが見つからない
```
⚠️ YOLOv8モデルが見つかりません
```
→ `app/src/main/assets/yolov8n.tflite`を確認

### ボールを検出できない
- 動画の明るさを確認
- ボールが小さすぎる場合は検出困難
- カメラアングルを調整

### 処理が遅い
- GPU対応デバイスを使用
- または`yolov8n.tflite`（最軽量）を使用
